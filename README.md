# practica-GitHub

Un repositorio de prueba para practicar y familiarizarse con las funcionalidades de Git y GitHub.

## Descripción

Este repositorio ha sido creado con el propósito de servir como un entorno seguro para practicar operaciones comunes de Git y explorar las características de GitHub. Es ideal para quienes están aprendiendo control de versiones o quieren experimentar con flujos de trabajo colaborativos.
